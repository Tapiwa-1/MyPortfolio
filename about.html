About Me

That moment when l started learning Information Technology and Communication as a Cambridge O level candidate, that is when l fall in love with technology. It gave the that eagerness of wanting to know the internet really function, what is the magic behind every software we use be it in mobile phones, Desktop Computer and even Web Application. I had that curiosity of wanting to know the hardware behind the functionality of a computer. With that subject ICT it created a path for me to become a developer.


We started with basic web scripting languages such as HTML and CSS, and learning Some Microsoft application such as Word, Excel, Presentation and even Microsoft access that latter on brought me the eagerness of learning Structured Query Language (SQL). I started creating basic static websites using HTML and CSS experimenting with properties such as display flex blocks etc.


I become an A level candidate and pick Computer Science as one of my subject. That subject revolutions the way l viewed the computer, it changed the whole of my mind-set. With fundamentals of problem solving where we used language like VB.Net, it gave me that determination of me wanting to be a software engineer regardless of what was going to happen. I when on to pick an under graduation degree Information Security And Assurance which is an extension of software engineering that mainly focus on protecting computer system and computer organisation. That is when l learned the dynamic part of websites with the use of Javascript (JS), PHP.


The unfortunate part of my career when l started my under graduate, c/c++ was my first actual programming language which for most is consider a very difficult language yet more powerful. I had to challenge that theory in so doing it introduced me to the fundamental of computer system, the study of the operating system as well as challenging data structures and algorithms (Linked List, stack, queues, trees etc). With this degree program we were introduced to .NetFrameword yet another very powerful framework where you can do everything with C#. I also learned database in conjunction with MySql to fulfil my dream of being a software/ Security Engineer. From there l started learning emerging technology such as ASP.NET, Tailwindcss, ReactJS because l am a true believer of self-learning together with being educated is equal to massive success.
